# pokedex-swiftui

Aplicación desarrollada en SwiftUI que consume la API pública PokeAPI para mostrar información de Pokémon.

## Tecnologías
- SwiftUI
- MVVM
- URLSession
- Async/Await

## Funcionalidades
- Listado de Pokémon
- Detalle de Pokémon
- Carga de imágenes
- Consumo de API REST

## Arquitectura
La app utiliza el patrón MVVM para separar lógica de presentación y manejo de datos.

## Instalación
1. Clonar el repositorio
2. Abrir el proyecto en Xcode
3. Ejecutar en simulador o dispositivo
